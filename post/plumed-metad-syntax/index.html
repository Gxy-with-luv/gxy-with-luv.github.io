关于PLUMED中BEMeta的语法

2020-09-19 2 min read # PLUMED # MetaD

PLUMED官网的示例使用Gromacs2018及以前的版本，使用-multi -replex等指定副本，所有也文件可以放在一个文件夹内，使用@replicas语法来指定不同副本的bias：

[/dir] ls
plumed.dat topol0.tpr topol1.tpr topol2.tpr topol3.tpr topol4.tpr structure.pdb
[/dir] vi plumed.dat
cv1: ...
cv2: ...
cv3: ...
cv4: ...
metad: METAD ...
  ARG=@replicas:{cv1,cv2,cv3,cv4}
  HEIGHT=@replicas:{?,?,?,?}
  PACE=?
  BIASFACTOR=?
  SIGMA=@replicas:{?,?,?,?}
  TEMP=?
  FILE=HILLS
...
[/dir] mpirun -np ?? gmx_mpi mdrun -v -plumed plumed.dat -multi 4 -replex ????? -nsteps ?????????

Plumed x gromacs2019及以后 (以2019.6为例，无-multi语法，全部改用-multidir)，每个副本文件夹内都需要plumed-common.dat与plumed.x.dat(其中需要include plumed-common.dat)和两个tpr文件：

[/dir] ls
rep0  rep1  rep2  rep3
[/dir/rep0] ls
plumed-common.dat plumed.0.dat topol.tpr topol0.tpr
[/dir/rep1] ls
plumed-common.dat plumed.1.dat topol.tpr topol1.tpr
...
[/dir/rep0] vi plumed-common.dat
cv1: ...
cv2: ...
cv3: ...
cv4: ...
#LOWER_WALLS ... (restraints)
#UPPER_WALLS ... (restraints)
[/dir/rep0] vi plumed.0.dat
INCLUDE FILE=plumed-common.dat
metad_cv1: METAD ARG=cv1 PACE=??? HEIGHT=? SIGMA=? BIASFACTOR=? TEMP=? FILE=HILLS
[/dir]
mpirun -np ?? gmx_mpi mdrun -v -multidir rep0 rep1 rep2 rep3 -s topol -deffnm ?? -plumed plumed.dat -replex ????? -nsteps ?????????
