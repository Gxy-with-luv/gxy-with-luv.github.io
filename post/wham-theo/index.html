数学推导

概率分布p_ij和p_j^{\circ}如果有以下关系：

pij=ficijpj∘p_{ij} = f_i c_{ij} p_j^{\circ} pij​=fi​cij​pj∘​

在数学上可以证明：

pj∘=∑i=1SNipij∑i=1SNificij=∑i=1Snij∑i=1SNificijp_{j}^{\circ}=\frac{\sum_{i=1}^{S} N_i p_{i j}}{\sum_{i=1}^{S} N_{i} f_{i} c_{i j}}=\frac{\sum_{i=1}^{S} n_{i j}}{\sum_{i=1}^{S} N_{i} f_{i} c_{i j}} pj∘​=∑i=1S​Ni​fi​cij​∑i=1S​Ni​pij​​=∑i=1S​Ni​fi​cij​∑i=1S​nij​​

p_j^{\circ}是我们想要的j构象处的unbiased概率，p_ij是我们第i个副本bias下的j构象处概率。

我们利用配分函数以及能量相差U_{bias:i}来建立p_j^{\circ}和p_ij关系：

pj∘=exp(−βEj∘)/∑jexp(−βEj∘)p_j^{\circ} = exp(-\beta E^{\circ}_j) / \sum_j exp(-\beta E^{\circ}_j) pj∘​=exp(−βEj∘​)/j∑​exp(−βEj∘​)

pij=exp(−βEij′)/∑jexp(−βEij′)=exp(−βEj∘−βUbias:ij)/∑jexp(−βEij′)p_{ij} = exp(-\beta E^{'}_{ij})/\sum_j exp(-\beta E^{'}_{ij}) = exp(-\beta E^{\circ}_{j}-\beta U_{bias:ij})/\sum_j exp(-\beta E^{'}_{ij}) pij​=exp(−βEij′​)/j∑​exp(−βEij′​)=exp(−βEj∘​−βUbias:ij​)/j∑​exp(−βEij′​)

分母都积分j与j无关，比值取为f_i:

fi=∑jexp(−βEij′)/∑jexp(−βEj∘)f_i =\sum_j exp(-\beta E^{'}_{ij})/\sum_j exp(-\beta E^{\circ}_j) fi​=j∑​exp(−βEij′​)/j∑​exp(−βEj∘​)

c_ij即:

cij=exp(−βUbias:ij)c_{ij} = exp(-\beta U_{bias:ij}) cij​=exp(−βUbias:ij​)

比较难解所以迭代😸，S个副本, J个bins的情况下我们有 len(f) = S 和 len(p_j^{\circ}) = J ，或者在不用bins的时候直接取J为已经采到的构型数，此时的p_j^{\circ}相当每帧的权重。

def get_wham_weights(nbias, nframes, bias_ts, nwham=NWHAM_, thres=THRES_):
    # find minimum bias
    min_bias = min(bias_ts)
    # initialize weights
    w = []
    for i in range(0, nframes): w.append(10**(-40))
    # offset and exponential of the bias
    expv = []
    for i in range(0, len(bias_ts)): expv.append(math.exp((-bias_ts[i]+min_bias)/KBT_))
    # c_ij get :D
    # initialize Z 
    Z = []
    for j in range(0, nbias): Z.append(1.0)
    # Z[] = 1/f_i ini=1
    # WHAM iterations
    for iii in range(0, nwham):
        print(iii)
        # store Z
        Z_old = Z[:]
        # recompute weights
        norm = 0.0
        for i in range(0, len(w)):
            ew = 0.0
            for j in range(0, len(Z)): ew += expv[i*len(Z)+j] / Z[j]
            w[i] = 1.0 / ew
            norm += w[i]
        # normalize weights
        for i in range(0, len(w)): w[i] /= norm
        # recompute Z
        for j in range(0, len(Z)): Z[j] = 0.0
        for i in range(0, len(w)):
            for j in range(0, len(Z)): Z[j] += w[i]*expv[i*len(Z)+j]
        # normalize Z 
        norm = sum(Z)
        for j in range(0, len(Z)): Z[j] /= norm
        # compute change in Z
        eps = 0.0
        for j in range(0, len(Z)): 
            d = math.log(Z[j]/Z_old[j])
            eps += d*d
        # check convergence
        if(eps<thres): break
    # return weights
    return w

观后感

  • 只要确定最后在新的势能上系统已经达到平衡态就可以了，概率只要能用配分函数写就可以满足证明~所以如果认为高斯势衰减比较快，以append方式再令加一个well-tempered bias也是可以的；
  • 恒定Bias因为一直走一直加就没有办法直接用这个方法处理，如果使用恒定Bias把Bias关掉后再采样到平衡然后如上处理应当也是没问题的。
